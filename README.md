Installing <testscenarios>
==================

Installing `<testscenarios>` from the main channel can be achieved by:

```
conda install <testscenarios>
```

It is possible to list all of the versions of `<testscenarios>` available on your platform with `conda`:

```
conda search <testscenarios>
```
